# Akademi Waitlist Static Site

This folder is a standalone static landing page for the Akademi beta waitlist.

## Render Static Site Settings

- Root Directory: `akademi-waitlist`
- Build Command: leave empty
- Publish Directory: `.`

The form posts waitlist submissions to:

`https://akademi-app.onrender.com/waitlist`

To point at another API host later, set `window.AKADEMI_API_URL` before loading `script.js`.
